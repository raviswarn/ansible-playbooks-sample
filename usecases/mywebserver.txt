- What needs to be done
  - install the apache web serve on the remote systems
  - create the web server directory
  - webserver directory has proper ownership
  - configuration of apache
    - copy the http configuration file
    - set up the vhost directory
    - add any SSL keys in needed
    - add the vhost template for the default directory
  - copy the site code to the new directory
  - start the apache service
  - add/enable any module needed
    - mysql/mariaDB
    - ssl
    - mod/rewrite
  - restart the apache service (or move the start here)
